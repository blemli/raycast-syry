# Syry for Raycast

Place calls from Raycast using the [`syry`](https://github.com/blemli/syry) CLI and a Yealink T31P phone.

## Commands

- **Call Number** — Enter a phone number and Raycast will trigger `syry call <number>`. Spaces, dashes, parentheses and a leading `(0)` are stripped automatically.

## Requirements

- `syry` must be installed and on `PATH` (or at `/Users/<you>/.local/bin/syry`).
